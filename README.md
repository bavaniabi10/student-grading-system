# Student Grading System - Backend (Express + MongoDB)

## Quick start

1. Copy `.env.example` to `.env` and update variables.
2. Install dependencies:
   ```
   npm install
   ```
3. Start (development):
   ```
   npm run dev
   ```
4. Start (production):
   ```
   npm start
   ```

## Available endpoints (basic)
- POST /api/auth/register
- POST /api/auth/login
- POST /api/grades
- GET /api/grades
- GET /api/analytics/summary
